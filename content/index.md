# Literature Notes

A minimal literature workspace for turning papers into durable notes and connecting them through topic synthesis.

## Start here

- Browse consolidated [paper notes](papers/).
- Browse synthesis [notes](notes/).
- Keep citation metadata in paper-note frontmatter and BibTeX entries in `bibliography.bib`.

Transient review streams are intentionally not committed as Markdown content. They should be rendered from GitHub-hosted discussions, issues, or comments in a future dynamic surface, similar in spirit to how Giscus embeds GitHub Discussions.
